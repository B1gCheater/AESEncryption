AES256 加密工具

一个基于AES256-CBC算法的图形化文件加密工具，提供安全可靠的文件加密保护。

特性

🔒 强加密算法: 使用AES256-CBC模式加密

🖥️ 友好界面: 基于PySide6的图形化操作界面

🔑 灵活密钥: 支持密码或直接密钥两种方式

🌍 多语言支持: 完全支持UTF-8编码，兼容各种语言字符

⚡ 高效性能: 使用Python cryptography原生加密库

系统要求:
操作系统: Windows 64位

Python环境: 需要安装相关依赖库

技术实现:
加密算法: AES256-CBC

密钥长度: 256位（32字节），16进制表示为64字符

密钥生成: 通过SHA256哈希将任意长度密码转换为固定长度密钥

UI框架: PySide6 + QT Designer

加密库: Python cryptography

安装依赖 pip install pyside6 cryptography


使用方法

重要警告:

⚠️ 加密前请注意：

如果不手动指定输出路径，程序将直接修改源文件

加密后请务必牢记密码或密钥，AES256加密无法破解

建议首次使用时先对测试文件进行加密解密操作


加密操作

运行程序，选择要加密的文件

输入加密密码（支持任意UTF-8字符）或直接输入64字符16进制密钥

选择输出路径（如不选择则直接修改原文件）

点击加密按钮


解密操作:

选择已加密的文件

输入加密时使用的密码或密钥

选择输出路径

点击解密按钮

密钥系统


密码模式

输入任意长度密码（支持各种语言符号）

程序自动通过SHA256转换为256位密钥

密码碰撞概率极低，确保唯一性


直接密钥模式

使用64字符16进制字符串作为密钥

格式示例：a1b2c3d4e5f6...（共64字符）

兼容性说明

即使丢失本程序，您仍然可以使用其他支持AES256-CBC算法的工具解密文件，只需确保：

使用相同的密钥或密码

使用相同的AES256-CBC算法


安全提醒
🔐 重要安全建议：

使用强密码（包含大小写字母、数字、特殊符号）

妥善备份密钥和密码

首次使用建议在小文件上测试

加密重要文件前确保已理解操作流程


技术细节

密钥生成：密钥 = SHA256(UTF-8编码的密码)

加密模式：AES256-CBC

密钥固定长度：32字节（256位）

16进制表示：64字符长度
